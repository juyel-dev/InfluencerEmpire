## Phase: Feature Development / Core Gameplay Loop

Goal: make the existing day-cycle loop engaging and legible.

Candidate work (pick by value, not volume):
- **End-of-Day summary** — recap of the day's gains (followers, money, activities) shown after End Day.
- **First-run onboarding** — explain the loop to new players.
- **Objective/goal tracker** — surface the next milestone progress in the HUD.
- **Activity result clarity** — richer feedback per activity (already has toasts; consider detail).

## Principles

- Product-first, but **protect the foundation**.
- One meaningful feature at a time; verify with the full gate suite.
- No architecture redesign without a measured justification.
